# 6ixFinance

Institutional-grade financial leadership for public companies listed on TSX-V, CSE, and NEO Exchange.

## Overview

6ixFinance delivers fractional CFO, controllership, and regulatory reporting services purpose-built for small-cap public companies. The site showcases SEDAR+ filing capabilities, NI 52-109 compliance, board-level reporting, and accounting system migration services.

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Framer Motion
- React Router v6
- Deployed on Netlify

## Pages

- **Home** — Hero, Stats, What We Do, Who We Serve, Process, Compliance Calendar, Testimonials
- **Services** — Fractional CFO, Controllership, SEDAR+ Filing, Board Reporting, FP&A, Accounting System Migration
- **About** — Story, Approach, Team
- **Contact** — Contact form + Booking

## Features

- Institutional-grade design with dark theme
- SEDAR+ compliance calendar
- Interactive service cards
- Testimonials carousel
- Fully responsive (mobile-first)
- SEO optimized

## Development

Built from scratch based on client brand guidelines and site architecture documentation. All design, content, and functionality tailored for public company financial leadership positioning.
